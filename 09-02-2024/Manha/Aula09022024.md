## O que é lematização?


| Palavras    | Lema        |
| ----------- | ----------- |
| Somos       |             |
| Era                Ser
| É 
| Sou

__Ser__ é a forma nominal onde não há quaisquer marcas de __tempo__, __modo__, __género__ e __número__.
